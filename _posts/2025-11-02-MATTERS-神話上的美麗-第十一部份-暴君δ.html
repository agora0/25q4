B 想了想，再點了一杯自己剛剛喝過的酒。

就在此時，B 聽到 A 那酒醉而睡的朋友發出的聲音。

X︰不…騙子…我不要喝你喝的這杯…美麗的…虛假的…一切都是浮影…所有都是夢幻…我就不喝你點的這杯…沒有任何的真實…不好喝…好喝…不好喝…好喝的不好喝…喝了不好喝的好喝…錨……酒…好喝的錨……不好喝的錨…….酒裏放錨…….我沒醉……是錨醉了…..海也醉了……錨喝醉了……錨喝了錨……

B 還以為這位 A 的朋友終於要醒來，卻發現 X 在說完這些斷斷續續不成章法的醉話後，又繼續不省人事，再陷入了安靜的睡眠中。

B︰沒想到你朋友不只長得好看，說話的聲音還挺好聽的。

A︰噢，的確，X 是一位很有魅力的人。

B︰X？這真是個奇特的名字，我從未聽過這個名字，這名字的發音很有趣，也很好聽。

A︰這不是 X 本來的名字，而是是 X 的藝名，按 X 的說法，X 是一個屬於戲劇 (δρᾶμα = drama) 的名字，而 X 不喜歡別人稱呼自己原來的真名。

B︰你這位朋友很有性格，那你知道 X 的真名嗎？

A︰X 曾經告訴過我自己的真名，但這令我想起一段困惑的經歷。

B︰甚麼經歷？

A︰X 總是喜歡拋出一些我解決不了的難題，讓我陷入迷茫與無助。

B︰噢？

A︰我的朋友，當 X 向我說完自己的真名後，向我拋出一個我至今無法回答的問題，X 問我，雖然我已知曉了一個「真名」，但又如何確定我不是被騙？可能這個所謂的「真名」也只是另一個「藝名」，是另一個舞台的名字，是另一個藝術創作的名字。

B︰這是朋友之間的玩笑？看來你們感情不錯。

A︰噢，雖然 X 的確是我的朋友，當 X 向我說這段話時，可是相當的生氣，也沒有開玩笑的意思。

B︰怎會這樣？你激怒了你的朋友？

A︰呀，我的朋友，為甚麼你會這樣想呢？我無意激怒任何人。

B︰哈哈，可能是長得好看的人聲音也好聽的人天然令人更容易信任？我總覺得你的朋友是被冒犯的一方。而且根據我的經驗，你那些奇怪的問題的確會令人產生不耐煩。

A︰神明在上，這些問題是我認真想知道答案，沒有想冒犯別人的意思。

B︰那 X 當時為甚麼生氣呢？

A︰噢，我的朋友，我也很想知道這個問題的答案，我只記得我說起一段神話後，X 的反應就很激烈，然後對我說了一大段話。

B︰X 說了甚麼呢？

A︰X 並對我說，X 是一個藝名，既然人能有藝名，那藝名是不是也可以擁有自己的藝名？反正都只是名字，都是人任意命名的，那到一個人如何能分辨到底哪一個才是真名呢？

B︰等等，我有點混亂，甚麼叫「藝名也可以擁有自己的藝名」？

A︰噢，我也問過 X 這個問題。面對這個問題，X 答我，X 是一個藝名，X 是一個名字 (ὄνομα = name)，這名字是 X 自己任意命名的。

B︰嗯，這我明白，然後呢？

A︰X 說，藝名可以不斷重覆這個任意命名的過程，藝名可以有自己的藝名，這是對藝名任意命名的任意命名，X 這個名字也能有自己的名字，這是名字的名字，是 X 的 X，X 的 X 可以是 Y，X 的 X 也可以是 Z。

B︰這太混亂了。

A︰X 解釋說，這是對命名的命名，或者是更一步，對命名的命名的命名，這種過程能無限推下去。

B︰我還是不明白，我能理解「名字是人任意的命名」，但人為甚麼要對一個名字再命名呢？這樣做有甚麼有價值的意義嗎？這對我來說沒有任何現實意義啊！這是一種單純的思想遊戲嗎？

A︰我的朋友，我能明白你的感受。

B︰我們叫「蘋果」做「蘋果」，這是我們對「蘋果」的命名，但「蘋果這個名字」我們並不需要再去命名，我們也不會這樣去做，所以也沒有「蘋果的名字的名字」。

B 看到 A 有點走神。

B︰你又在想甚麼呢？

A︰噢，我剛才腦海中閃過一些奇怪的畫面。

B︰甚麼畫面？

A︰我好像看到一個金色的蘋果在對我笑，它溫和地對我說，我親愛的良朋啊，你一直稱呼我為金色的蘋果，但那只是我為了你所取的藝名，我的真實名字你無法翻譯，那你是不是想問，我的名字到底是甚麼？我的朋友啊，這不是我一直在問你的問題嗎？你的名字是甚麼呢？

B︰我的天，我看你也醉了，你不會撐不了多久也倒在桌上了吧？

A︰很抱歉我的朋友，我也不確定我有沒有醉。或者我已經醉了，我現在似乎無法脫離那個金色蘋果的問題，我一直在想「你的名字是甚麼」「我的名字是甚麼」這些問題。

B 看上去很頭痛。

B︰別想了！你不能這樣思維跳躍！等等，我都要被你搞瘋了，我們之前在說甚麼來著？

A︰就是若果一個蘋果有名字，那這個蘋果的名字有沒有自己名字。

B︰沒有！蘋果的名字沒有名字，若果你問一個蘋果它的名字是甚麼，蘋果會對你說它叫蘋果，但若你問一個蘋果它的名字叫甚麼名字，它恐怕會看傻子一樣看著你，覺得你是一種特別愚蠢的水果！

A︰但是，我的朋友，我不是水果啊！

B 笑了起來。

B︰你還能記得你不是水果就好，我們能不能回到正常點的話題上去了？

A︰噢，我的朋友，我似乎找到為甚麼我會聯想到金色蘋果畫面的線索。

B︰希望你這次不會聯想到更離譜的東西。

A︰保證不會，我終於記起我的朋友 X 曾被甚麼神話所激怒，那個神話中，有一個金色的果子。

B︰甚麼神話？

於是 A 向 B 回憶起那些金色果子的神話。

很久很久之前，在一個美麗的小島上，有一對相愛的夫妻，他們誕下了一個孩子。

他們感謝神明的賜與他們新的生命，也感謝神明對他們的愛的守護。

然而相愛的兩人，卻一直沒想好新生的孩子叫甚麼。

他們想了很多名字，想了又想，想了又想。

他們以為孩子出生後就能定下來，卻沒想到孩子出生都過了一個月了，孩子的名字到底是甚麼還沒想好。

有一天，他們逗弄新生的小嬰兒，對只會哭的小嬰兒說，我們的孩子啊，你想被稱為甚麼呢？你想將來成為甚麼呢？你想被人們知曉為甚麼呢？我們的小寶貝，你只會哭，不如就叫你小哭包吧！

二人無心逗趣之語，卻沒想到成為了現實，這個新生的孩子於是從那天後就沒停過哭，哭了足足 3 天 3 夜。

儘管小嬰兒看上去還是相當的健康，這個反常的現象讓夫妻陷入了恐慌，於是他們決定向神明求助，希望神明能讓他們的孩子的哭泣停下來。

他們準備好獻給神明的祭品，帶著孩子坐船到神明神廟所在的小島，將船停泊好後，在神廟誠心的祈求。

似乎是神明聆聽到夫妻二人的禱告，在當天晚上，他們做了一個夢。

在夢中，一個金色發光會飛的 果子 (καρπός = fruit, result) 笑著對他們問，你們既然相愛，那你們知道必定知道我的真實名字，我的名字是甚麼呢？

夫妻二人猜了很多答案，都沒有答對。

這些答案中有「金色的蘋果」，「發光的蘋果」，「珍貴的蘋果」，「金色發光的蘋果」，「珍貴發光的蘋果」等等等等。

金色的果子對他們說，那可能是我在 戲劇 (δρᾶμα = drama) 中的名字，我很喜歡這些名字，但我的真實名字是甚麼呢？我到底是甚麼 果子 (καρπός = fruit, result)？

兩夫婦在夢境中再猜了很多了次，卻都沒有猜中。

這令他們在夢中也急了起來。

急著急著他們就醒了。

或者是神明真的介入了他們的困境，醒來後他們發現他們的孩子不哭了，他們看到小嬰兒抱著一個金色果子安穩地睡著。

於是，他們的孩子就被取名為「金色的蘋果」。

B︰這就沒了？好短的神話，這我就不能理解了，我的朋友。

A︰我的朋友，你不理解甚麼呢？

B︰我不理解為何 X 會被你這個神話所激怒。

A︰我也一直在想這個問題呢。

B︰你再想想，你沒有說別的東西？

A︰噢，我記起了，這個神話還有下半段。

B 露出相當無奈的表情，喝了一口酒，示意 A 繼續說下去。

又很多很多年後，這個被取名為「金色的蘋果」的孩子長大成人。

有一天，這個孩子在家附近另一個小島上，遇到一個老人。

一大一小聊了起來。

老人︰孩子，你的名字是甚麼呢？

孩子︰我叫金色的蘋果。

老人似乎很意外孩子有這個名字，就問起這個名字的來由，孩子於是將自己爸媽經歷過的故事講給老人聽。

令孩子意外的是，老人之後就一直在笑，笑得停不下來。

孩子︰老人家，我不明白，我的名字真有這麼好笑嗎？

老人︰抱歉，孩子，我真的沒想笑，我只是忍不住。

孩子︰我不明白。

老人︰是我的錯，在我的家鄉裏，那裏的人都喜歡種果子，種果子甚至成為了一種正式的賽事，每年最甜美的果子，都會奉獻給神明。

孩子︰噢。

老人︰我家鄉這種文化風俗甚至影響到孩子的取名，就像我的爸媽，把我的名字取成「上天的甜美」。但是，我一點都不喜歡這個名字，我的天，孩子，你大概不明白這個名字從小到大曾讓我受到多少人的取笑。

孩子︰噢。

老人︰在我的家鄉裏，最甜美的果子的名字是「蘋果」，就是你的名字的來由。

孩子︰原來是這樣嗎？

老人似乎充滿歉意。

老人︰是的。但是啊，孩子，其實你的家鄉實際上沒有任何真正的蘋果。

孩子︰啊？

老人︰在我年輕時，在一次無聊的航行中，我想到了一個惡作劇。我把我家鄉最不甜美、最清淡無味的果子帶到這幾座小島並種下。當這裏的人們問起我，這是甚麼果子，我就和別人說，這是一種金色果子，叫蘋果，它相當的甜美。這裏的人相信了，並將這種不甜美、沒有甚麼味道的果子稱為蘋果。

孩子︰啊？

老人︰然而這個世上沒有金色的蘋果，這種果子從來也不甜美。

孩子︰啊？所以，我從小吃到大吃的果子不是叫蘋果？

老人︰很抱歉，我沒想到會影響到你，我年輕時只是想開一個玩笑。

孩子︰但這些果子真的很甜啊。

老人︰孩子，這就是問題所在，我萬萬沒有想到，這種果子種在你的家鄉後，變甜了。我覺得，這真是命運女神對我的惡作劇，是命運女神對我的嘲笑。你很難想像當我知道這裏的人告訴我果子真的很甜時，我有多大的震驚。

孩子︰老人家，那我想知道，這種金色果子真正的名字是甚麼。

老人︰為甚麼你想知道呢？

孩子很認真的看著老人，誠心的說︰因為我想知道我真實的名字，我到底叫甚麼。

老人想了想，也不知道怎回答孩子的問題。

老人並不想這個孩子而此改名。

老人︰孩子，在你的家鄉，這種甜美的金色果子就叫蘋果，金色的蘋果是存在的。

孩子陷入了茫然。

B 見 A 又停了下來，沒再說話。

B︰所以那種金色的果子到底叫甚麼？

A︰我忘記了，我忘記了之後這神話的內容了。

B︰我覺得我也要像 X 一樣生氣了，你不能說故事說一半就忘記了後面的內容。

A︰很抱歉，我的朋友。
